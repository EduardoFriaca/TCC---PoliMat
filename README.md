# TCC---PoliMat
Sistema educacional em Python para apoiar o ensino de polinômios no Ensino Médio, com interface em Kivy e manipulação algébrica usando SymPy.


## Como baixar o aplicativo pronto

Para usar o PoliMat sem instalar Python:

1. Acesse a aba **Releases** deste repositório.
2. Baixe o arquivo **PoliMat_App.zip**.
3. Extraia a pasta no computador.
4. Abra a pasta extraída.
5. Clique duas vezes em **PoliMat.exe**.

Não é necessário instalar Python para usar a versão executável.
